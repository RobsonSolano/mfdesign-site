## Anti-Patterns Verdict

LLM: gradient roxo→rosa, cards idênticos, hero "Hello, I'm X" com SVG dots, Plus Jakarta+Bootstrap = SaaS lookalike. Forte sinal de template/AI.

Detector: 6 warnings — 3× overused-font (Plus Jakarta), 3× single-font.

## Priority Issues

[P0] Gradiente roxo→rosa em todo lugar — anti-referência principal do PRODUCT.md. Fix: remover .bg-gradient-primary-to-secondary, .text-gradient, bg-cta-*.png; usar Atelier Gold + neutros warm. Suggested: colorize.

[P0] Hierarquia tipográfica plana — só Plus Jakarta, sem display/body distintos. Fix: introduzir display serif (Cormorant/Tenor/DM Serif, evitar Fraunces overused) + body sans. Suggested: typeset.

[P1] Cards uniformes 2×3 e 1×6 — antipattern "identical card grids". Fix: variar tamanhos, adicionar Gallery Plate metadata (categoria UPPER gold + nome + contexto). Suggested: layout.

[P1] Hero compete com o trabalho — meia tela antes de UM projeto. Fix: distill hero (texto reduzido, sem SVG dots, CTA "Ver galeria"). Suggested: distill.

[P2] Form Bootstrap padrão — DESIGN.md spec pede Underline Input (bottom-line only). Fix: reescrever fields, label uppercase tracked, erros em gold-deep. Suggested: polish.

## Persona Red Flags

Cliente de evento pessoal: gradiente eletrônico não combina com tom emocional, bounce em mobile.
Pequeno empreendedor: link Behance genérico (behance.net/) quebra confiança em 1 clique.
Empresa formal: hero "Hello, I'm" + foto grande classifica como "freelancer iniciante".
